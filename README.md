A weather application built using HTML, CSS, and JavaScript.
It uses OpenWeather API and displays animations like sun, clouds,
rain, and night mode based on weather conditions.

## Features
- Live weather data
- Sun / Cloud / Rain animations
- Night mode (moon & stars)
- Responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API

## How to Run
1. Open index.html in browser
2. Add your OpenWeather API key
3. Enter city name
